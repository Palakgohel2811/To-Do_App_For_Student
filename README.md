# 📋 Task Manager Web App

A full-stack Task Manager web application built using Flask. This app allows users to manage their daily tasks efficiently with features like authentication, task tracking, and a clean user interface.

---

## 🚀 Features

- 🔐 User Registration & Login
- ✅ Add, Delete, and Mark Tasks as Completed
- 📅 Add Due Dates to Tasks
- 🗂 Categorize Tasks (Study / Personal)
- 🔍 Search Tasks
- 🌙 Dark Mode Toggle
- 💾 Data stored using SQLite Database (per user)

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS
- **Database:** SQLite
- **Authentication:** Flask-Login
